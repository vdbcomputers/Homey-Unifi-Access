De mogelijkheid om gegevens te ontvangen van de Unifi Access API breidt de integratie uit met het Unifi Access-ecosysteem. Momenteel is het alleen mogelijk om apparaatmeldingen te ontvangen, maar deze meldingen worden opgeslagen als bruikbare gegevens die verder kunnen worden verwerkt of getriggerd in flows binnen het systeem. Dit betekent dat zodra er een melding of notificatie van een Unifi-apparaat binnenkomt, deze informatie automatisch wordt opgeslagen en beschikbaar wordt gesteld voor verdere acties, analyses of rapportages.

Belangrijke Functionaliteiten
Ontvangen van Apparaatmeldingen:

De API ontvangt notificaties van Unifi Access-apparaten. Dit kan bijvoorbeeld een melding zijn van een deur die wordt geopend, een kaartlezer die wordt geactiveerd, of een foutmelding van een aangesloten Unifi-apparaat.
De meldingen worden in real-time ontvangen, waardoor het systeem altijd up-to-date is met de laatste meldingen van de apparaten.
Data Opslag en Beschikbaarheid in Flows:

Alle ontvangen notificaties worden opgeslagen als data-entiteiten. Deze gegevens zijn direct toegankelijk voor gebruik in geautomatiseerde processen.
Gebruikers kunnen specifieke acties of waarschuwingen instellen in reactie op deze meldingen, zoals een melding sturen naar een beveiligingsteam bij een deur-open event, of het loggen van alle gebeurtenissen voor toekomstige referentie.
